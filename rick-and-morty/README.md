## Getting Started

First, run the install scripts

`yarn install or npm  install`

once the installation is done

For production build

`yarn build`

or

`npm run build`

start the app

`yarn start`

or

` npm run start`

For Dev mode:

`yarn dev`
or `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
